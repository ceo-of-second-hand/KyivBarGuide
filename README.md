# kyiv-bar-guide

веб-додаток для пошуку та управління інформацією про бари києва. реалізовано на asp.net mvc з використанням entity framework.

основна функціональність включає перегляд каталогу барів, створення резервацій, додавання улюблених місць, перегляд меню та коктейлів. для барів доступна адміністративна панель з можливістю управління інформацією про заклад, меню та перегляду резервацій.

система підтримує ролі користувачів (клієнти та адміністратори барів), має інтеграцію з google maps для відображення локацій та включає систему рейтингів та відгуків.
